12.94 emacspeak-haskell

Speech-enable package haskell-mode

12.94.1 emacspeak-haskell Options

Variable: User Option voice-haskell-interactive-face-compile-error

Personality used for haskell-interactive-face-compile-error This personality uses voice-warning whose effect can be changed globally by customizing voice-warning-settings.

Variable: User Option voice-haskell-interactive-face-compile-warning

Personality used for haskell-interactive-face-compile-warning This personality uses voice-warning whose effect can be changed globally by customizing voice-warning-settings.

Variable: User Option voice-haskell-interactive-face-garbage

Personality used for haskell-interactive-face-garbage This personality uses voice-monotone-light whose effect can be changed globally by customizing voice-monotone-light-settings.

Variable: User Option voice-haskell-interactive-face-prompt

Personality used for haskell-interactive-face-prompt This personality uses voice-brighten whose effect can be changed globally by customizing voice-brighten-settings.

Variable: User Option voice-haskell-interactive-face-prompt-cont

Personality used for haskell-interactive-face-prompt-cont This personality uses voice-brighten-extra whose effect can be changed globally by customizing voice-brighten-extra-settings.

Variable: User Option voice-haskell-interactive-face-result

Personality used for haskell-interactive-face-result This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.
