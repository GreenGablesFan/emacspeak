12.128 emacspeak-nxml

nxml-mode is a new XML mode for emacs by James Clark. http://www.thaiopensource.com/download/

12.128.1 emacspeak-nxml Commands

12.128.1.1 emacspeak-nxml-summarize-outline

Command: emacspeak-nxml-summarize-outline
        Intelligent spoken display of current outline entry.

12.128.2 emacspeak-nxml Options

Variable: User Option emacspeak-nxml-attribute-colon-voice

Personality used for nxml-attribute-colon This personality uses voice-monotone whose effect can be changed globally by customizing voice-monotone-settings.
