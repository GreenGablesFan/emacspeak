12.108 emacspeak-kite

kite connects Emacs to browsers that support the webkit debugging protocol. You can use this to connect to a running Chrome Make sure to start chrome with the correct command-line flag: e.g., on Linux: google-chrome –remote-debugging-port=9222 You can get Emacs package kite from here: git clone git://github.com/jscheid/kite Make sure to first install the websocket package from elpa. This package speech-enables kite for Emacspeak users. This is what I use at present when developing/debugging ChromeVox. ChromeVox == http://google-axs-chrome.googlecode.com http://chromevox.com
