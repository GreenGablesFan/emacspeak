12.175 emacspeak-slime

;;; SLIME == Superior Lisp Interaction Mode For Emacs

Slime is a powerful IDE for developing in Common Lisp and Clojure. It’s similar but more modern than package ILisp that I used as a graduate student when developing AsTeR.

12.175.1 emacspeak-slime Options

Variable: User Option voice-slime-apropos-label

Personality used for slime-apropos-label This personality uses voice-monotone-light whose effect can be changed globally by customizing voice-monotone-light-settings.

Variable: User Option voice-slime-apropos-symbol

Personality used for slime-apropos-symbol This personality uses voice-monotone-light whose effect can be changed globally by customizing voice-monotone-light-settings.
