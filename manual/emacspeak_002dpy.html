12.144 emacspeak-py

This speech-enables python-mode available on sourceforge and ELPA

12.144.1 emacspeak-py Options

Variable: User Option emacspeak-py-XXX-tag-personality

Personality used for py-XXX-tag-face This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.

Variable: User Option emacspeak-py-builtins-personality

Personality used for py-builtins-face This personality uses voice-bolden whose effect can be changed globally by customizing voice-bolden-settings.

Variable: User Option emacspeak-py-class-name-personality
