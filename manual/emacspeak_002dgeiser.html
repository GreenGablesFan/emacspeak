12.82 emacspeak-geiser

geiser.el — GNU Emacs and Scheme talk to each other This module speech-enables all interactive aspects of geiser, including the geiser->scheme REPL. This is used by racket-mode for racket interaction, And also for interacting with Guile.

12.82.1 emacspeak-geiser Options

Variable: User Option emacspeak-geiser-voice-lock-autodoc-current-arg

Personality used for geiser-font-lock-autodoc-current-arg This personality uses voice-bolden whose effect can be changed globally by customizing voice-bolden-settings.
