12.104 emacspeak-jdee

Speech enable Java IDE. The Java IDE –JDEE– can be found at http://sunsite.auc.dk/jdee/

12.104.1 emacspeak-jdee Options

Variable: User Option emacspeak-jdee-java-voice-lock-api-personality

Personality used for jdee-java-font-lock-api-face This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.

Variable: User Option emacspeak-jdee-java-voice-lock-bold-personality

Personality used for jdee-java-font-lock-bold-face This personality uses voice-bolden whose effect can be changed globally by customizing voice-bolden-settings.

Variable: User Option emacspeak-jdee-java-voice-lock-code-personality

Personality used for jdee-java-font-lock-code-face This personality uses voice-monotone whose effect can be changed globally by customizing voice-monotone-settings.

Personality used for jdee-java-font-lock-constant-face This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.

Variable: User Option emacspeak-jdee-java-voice-lock-italic-personality

Personality used for jdee-java-font-lock-italic-face This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.

Variable: User Option emacspeak-jdee-java-voice-lock-number-personality

Personality used for jdee-java-font-lock-number-face This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.

Variable: User Option emacspeak-jdee-java-voice-lock-operator-personality

Variable: User Option emacspeak-jdee-java-voice-lock-pre-personality
