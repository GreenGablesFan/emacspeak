12.9 emacspeak

Emacspeak extends Emacs to be a fully functional audio desktop. This is the main emacspeak module. It actually does very little: It sets up Emacs to load package-specific Emacspeak modules as each package is loaded. It implements function emacspeak which loads the rest of the system.

12.9.1 Emacspeak Commands

12.9.1.1 emacspeak-describe-emacspeak

Command: emacspeak-describe-emacspeak
C-h C-e
<f1> C-e
<help> C-e
Give a brief overview of emacspeak.

12.9.1.2 emacspeak-submit-bug

Command: emacspeak-submit-bug
C-e C-M-b
<fn> C-M-b
Function to submit a bug to the programs maintainer.
