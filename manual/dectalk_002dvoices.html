12.4 dectalk-voices

This module defines the various voices used in voice-lock mode. This module is Dectalk specific.

12.4.1 dectalk-voices Commands

12.4.1.1 dectalk

Command: dectalk
C-e d C-d
<fn> d C-d
Select Dectalk TTS server.

12.4.1.2 dtk-exp

Command: dtk-exp
Select Dectalk TTS server.

12.4.2 dectalk-voices Options

Variable: User Option dectalk-default-speech-rate

*Default speech rate at which TTS is started.
