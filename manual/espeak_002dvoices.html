12.212 espeak-voices

This module defines the various voices used in voice-lock mode by the ESpeak TTS engine.

12.212.1 espeak-voices Commands

12.212.1.1 espeak

Command: espeak
C-e d C-e
<fn> d C-e
Start ESpeak engine.

12.212.2 espeak-voices Options

Variable: User Option espeak-default-speech-rate
