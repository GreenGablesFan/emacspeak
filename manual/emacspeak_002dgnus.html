12.85 emacspeak-gnus

This module advises gnus to speak. Updating support in 2014 (Emacspeak is nearly 20 years old)

12.85.1 emacspeak-gnus Commands

12.85.1.1 emacspeak-gnus-async

Command: emacspeak-gnus-async
C-, g
C-x @ a g
Run gnus on a separate thread.

12.85.1.2 emacspeak-gnus-summary-catchup-quietly-and-exit

Command: emacspeak-gnus-summary-catchup-quietly-and-exit
        Catch up on all articles in current group.

12.85.1.3 emacspeak-gnus-summary-hide-all-headers

Command: emacspeak-gnus-summary-hide-all-headers
Hide all headers in the article.
Use this command if you don’t want to listen to any article headers when
reading news.

12.85.1.4 emacspeak-gnus-summary-show-some-headers

Command: emacspeak-gnus-summary-show-some-headers
        Show only the important article headers,
i.e. sender name, and subject.

12.85.2 emacspeak-gnus Options

Variable: User Option emacspeak-gnus-large-article

*Articles having more than emacspeak-gnus-large-article lines will be considered to be a large article. A large article is not spoken all at once; instead you hear only the first screenful.

Variable: User Option emacspeak-gnus-punctuation-mode

Pronunciation mode to use for gnus buffers.
